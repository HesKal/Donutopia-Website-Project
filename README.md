This repository contains the source code for "Donutopia", a responsive website for a fictional donut shop. This project was my introduction to web technologies, built collaboratively as part of the "Web Technologies (CIS1203)" course.

## üöÄ Project Overview

The goal was to design and build a complete, user-friendly website for a new brand, "Donutopia". We followed a structured design process, from competitive analysis to final implementation, focusing on creating a visually appealing and consistent user experience.

### Key Features:
- **Responsive Design:** The website is fully functional on desktops, tablets, and mobile devices.
- **Multi-Page Layout:** Includes Home, About Us, Menu, Reviews, and Contact pages.
- **Interactive Forms:** A functional login page and contact form.
- **Brand Identity:** A consistent theme with a custom logo, color scheme, and typography.

## üé® Design & Planning Process

Before writing a single line of code, our team conducted a thorough planning phase:

1.  **Competitive Analysis:** We evaluated competitor websites like Dunkin' Donuts and Krispy Kreme to understand industry standards and identify opportunities.
2.  **Site Mapping & Wireframing:** We planned the website's structure and the layout of each page to ensure a logical user flow.
3.  **Design Principles:** We applied principles of contrast, alignment, repetition, and proximity to create a clean and cohesive design.
